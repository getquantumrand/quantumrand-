# QuantumRand

True quantum randomness as a service. Every number generated by this API is decided by quantum physics — not mathematical formulas.

**Live API:** [quantumrand.dev](https://quantumrand.dev)
**Docs:** [/docs](https://quantumrand.dev/docs)

## Quick Start

### 1. Get a free API key

Sign up at [quantumrand.dev/signup](https://quantumrand.dev/signup) to get your API key instantly.

### 2. Generate quantum random numbers

```bash
# Random bits
curl -H "X-API-Key: qr_your_key" \
  "https://quantumrand.dev/generate/bits?n=256"

# Random integer
curl -H "X-API-Key: qr_your_key" \
  "https://quantumrand.dev/generate/integer?min=1&max=1000"

# Cryptographic key
curl -X POST -H "X-API-Key: qr_your_key" \
  "https://quantumrand.dev/generate/key?bits=256"
```

## SDKs

### Python

```bash
pip install quantumrand
```

```python
from quantumrand import QuantumRandClient

qr = QuantumRandClient("qr_your_key")
bits = qr.bits(256)       # "10110010..."
number = qr.integer(1, 100)  # 42
key = qr.key(256)          # "a3f9c2e1..."
```

### JavaScript

```bash
npm install quantumrand
```

```javascript
const { QuantumRandClient } = require('quantumrand')

const qr = new QuantumRandClient({ apiKey: 'qr_your_key' })
const bits = await qr.bits(256)
const number = await qr.integer(1, 100)
const key = await qr.key(256)
```

## Quantum Backends

| Backend | Type | Speed |
|---------|------|-------|
| `aer_simulator` | Qiskit local simulator | Instant |
| `origin_cloud` | Origin Quantum cloud simulator | ~1s |
| `origin_wuyuan` | Origin Quantum real hardware | ~5s |
| `ibm_hardware` | IBM Quantum real hardware | 2-30 min (queued) |

Pass `?backend=aer_simulator` for instant results or `?backend=ibm_hardware` for bits from a real quantum computer.

## How It Works

1. **Superposition** — Qubits enter superposition via Hadamard gates, existing as 0 and 1 simultaneously
2. **Measurement** — Quantum measurement collapses the state; physics alone decides the outcome
3. **Your Response** — Raw quantum bits are formatted into bits, hex, integers, or cryptographic keys

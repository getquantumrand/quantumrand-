Privacy Policy

Last updated: March 19, 2026

1. Information We Collect

When you use QuantumRand, we collect the following information:

  • Account information: Name and email address provided during API key creation.
  • Usage data: API endpoint called, number of bits requested, response time, and timestamp. This is linked to a hash of your API key, not the key itself.
  • Technical data: IP address (for rate limiting only, not stored long-term).

2. Information We Do NOT Collect

  • We do not store your plaintext API key. Only a SHA-256 hash is retained.
  • We do not store the random numbers generated for you. Output is returned in the response and immediately discarded.
  • We do not use cookies or tracking pixels.
  • We do not sell, rent, or share your personal information with third parties.

3. How We Use Your Information

  • To authenticate API requests and enforce rate limits.
  • To provide usage statistics via the /keys/stats endpoint.
  • To monitor service health and investigate abuse.
  • To communicate important service updates (e.g., breaking changes, security notices).

4. Data Storage and Security

Data is stored in Google Firebase Firestore. API keys are hashed with SHA-256 before storage. We use HTTPS for all API communication. Access to production systems is restricted to authorized personnel.

5. Data Retention

  • Account data: Retained until you request deletion or your key is revoked.
  • Usage logs: Retained for up to 90 days for analytics, then automatically purged.

6. Your Rights

You may request:

  • A copy of the data we hold about your API key.
  • Deletion of your account and associated usage data.
  • Correction of inaccurate account information.

To exercise these rights, contact us at privacy@quantumrand.io.

7. Third-Party Services

We use the following third-party services:

  • Google Firebase: Database hosting (see Google's privacy policy).
  • Railway: Application hosting.
  • IBM Quantum: Quantum hardware access (when ibm_hardware backend is selected).
  • Sentry: Error monitoring (no PII is sent).

8. Changes to This Policy

We may update this policy from time to time. Material changes will be communicated via the API or email. Continued use after changes constitutes acceptance.

9. Contact

For privacy-related questions, contact us at privacy@quantumrand.io.

2. Description of Service

QuantumRand provides a quantum entropy API that delivers true random numbers and cryptographic primitives generated using IBM Quantum hardware. The Service includes:

  • Core entropy endpoints (hex, integer, float, bytes, UUID)
  • Financial primitives (transaction IDs, OTPs, nonces, keypairs, audit signatures)
  • Industry-specific endpoints for fintech, gaming, healthcare, legal, cybersecurity, and IoT applications
  • Audit logging and compliance dashboard
  • SDKs for Python, JavaScript, and Go

The Service is provided on a subscription basis with a free tier and paid tiers as described at quantumrand.dev/pricing.
